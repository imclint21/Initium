# Initium Documentation

Initium is a library for simplifying .NET API development, offering standardized service operations, flexible routing, and seamless chaining of service results for cleaner and more maintainable code.

## Table of Contents

- [Getting Started](getting-started.md) — Installation, setup, and your first API
- [Service Results](service-results.md) — `ServiceResult`, chaining, typed results, and error handling
- [Controllers & Attributes](controllers.md) — `ApiController`, response attributes, validation, pagination, and custom headers
- [Examples](examples.md) — Full working examples covering common scenarios
